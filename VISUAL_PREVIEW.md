### Before Loading

```
┌─────────────────────────────────┐
│ 📭 Belum Ada Data               │
│                                 │
│ Pilih filter dan klik           │
│ "Muat Data Siswa" untuk memulai │
└─────────────────────────────────┘
```

### While Loading

```
┌─────────────────────────────────┐
│ ⏳ Memuat data...               │
└─────────────────────────────────┘
```

### With Data (Search)

```
[🔍 Cari: "Ahmad"]

Ditemukan: 2 siswa
├─ Ahmad Rizki (1234)
└─ Ahmad Subhan (1245)
```

### Empty Search Result

```
[🔍 Cari: "Xyz"]

┌──────────────────────────┐
│ 🔍 Tidak ada siswa yang  │
│    cocok dengan pencarian│
└──────────────────────────┘
```

## Responsive Breakpoints

```
Mobile:  0px - 767px    (md:hidden)
Tablet:  768px - 1023px (default)
Desktop: 1024px+        (hidden md:block)
```
